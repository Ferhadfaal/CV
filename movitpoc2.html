Moovit P1 PoC - WebView Intent Redirection

Impact: Critical (P1) - Arbitrary Activity Launch + Sensitive Data Exposure

Disclaimer: این PoC فقط برای演示 مسئولانه و Bug Bounty Moovit ساخته شده.
هیچ داده واقعی کاربر جمع‌آوری نمی‌شود. فقط intent scheme تست می‌شود.

نحوه تست:

  1. این فایل را روی localhost یا GitHub Pages بگذار (python3 -m http.server 8080)
  2. با ADB یا مرورگر بزن:
    moovit://webview?url=https://your-github-link/movitpoc.html
  3. دکمه را بزن → PaymentAccountActivity باز می‌شه (داده کارت اعتباری/حساب)

این PoC دقیقاً همان چیزیه که Cursor Critical اعلام کرد.
