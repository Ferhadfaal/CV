WebView runtime probe

Collects only what JavaScript can read in this page context (no hardcoded user secrets).

Note: This cannot read Moovit internal storage, accounts, or tickets. It only reads browser/WebView-exposed values (URL, query string, navigator, screen). To pass values in, append them to this page URL as query parameters, e.g. ?demo=1 — they will appear under hrefQueryParams.
